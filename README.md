# Task Management Application

A production-ready full-stack task management application built as part of a Full Stack Developer take-home assignment.

## Tech Stack

### Frontend

- React
- TypeScript
- Redux Toolkit
- React Router
- TanStack Query
- Tailwind CSS
- shadcn/ui

### Backend

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication

### DevOps

- Docker
- GitHub Actions
- Swagger

## Features

- User Authentication
- Task CRUD
- Dashboard Analytics
- Task Filtering & Sorting
- Kanban Board
- Dark Mode
- Real-Time Updates
- API Documentation
- Automated Testing
